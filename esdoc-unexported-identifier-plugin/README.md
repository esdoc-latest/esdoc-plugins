# ESDoc Unexported Identifier Plugin
## Install
```bash
npm install esdoc-latest-unexported-identifier-plugin
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdoc-latest-unexported-identifier-plugin", "option": {"enable": false}}
  ]
}
```

`enable` is default `false`.
