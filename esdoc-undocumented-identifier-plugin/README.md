# ESDoc Undocumented Identifier Plugin
## Install
```bash
npm install esdoc-latest-undocumented-identifier-plugin
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdoc-latest-undocumented-identifier-plugin", "option": {"enable": true}}
  ]
}
```

`enable` is default `true`.
